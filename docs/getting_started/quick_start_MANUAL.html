Quick Start Guide

This guide will walk you through creating your first MCHP Blockset project in 10 minutes.

Step 1: Open Simulink Library Browser

  1. Launch MATLAB
  2. Type simulink in the command window and press Enter
  3. In the Library Browser, find MCHP Blockset

Step 2: Create a New Model

  1. Click File → New → Model (or Ctrl+N)
  2. Save the model as my_first_mchp_project.slx

Step 3: Configure Target Hardware

  1. Open Model Configuration Parameters (Ctrl+E)
  2. Navigate to Code Generation → System target file
  3. Select one of the MCHP target files:
    • MCHP_dsPIC_PIC24.tlc - For dsPIC/PIC24 devices
    • MCHP_PIC32.tlc - For PIC32 devices
    • MCHP_SAM.tlc - For SAM devices
  4. Click Apply

Step 4: Select Device and Compiler

  1. In Configuration Parameters, go to Code Generation → MCHP Target
  2. Select your target device (e.g., dsPIC33AK128MC106)
  3. Select compiler version (auto-detected if installed correctly)
  4. Click OK
Tip: Use the search box to quickly find your device in the list.

Step 5: Add Master Block

  1. From MCHP Blockset library, drag Master block to your model
  2. Double-click to configure:
    • Set system clock frequency
    • Configure oscillator settings
    • Set sample time (e.g., 0.0001 for 100μs)
Important: The Master block is required in every MCHP Blockset model. It initializes the device and configures the system clock.

Step 6: Add Peripheral Blocks

Example: Toggle a Digital Output

  1. Add Digital Output block from MCHP Blockset → GPIO
  2. Double-click the block:
    • Select output pin (e.g., RB0)
    • Set initial value (0 or 1)
  3. Add a Pulse Generator from Simulink → Sources
  4. Connect Pulse Generator output to Digital Output input
  5. Configure Pulse Generator:
    • Period: 2 (toggle every second)
    • Sample time: Inherited (-1)

Step 7: Build the Model

  1. Click the Build button (or press Ctrl+B)
  2. Wait for code generation and compilation
  3. Check for successful build in the Diagnostic Viewer
  4. Verify output files are created:
    • *.hex - Programming file
    • *.elf - Executable with debug symbols

Step 8: Program the Device

  1. Option A: Using MPLAB X IPE
    • Open MPLAB X IPE
    • Select device and programmer
    • Browse to .hex file in model folder
    • Click "Program"
  2. Option B: From Simulink (if configured)
    • In model configuration, enable auto-programming
    • Click "Build Model" - device will be programmed automatically

Example Models

The blockset includes example models to help you get started:

  • Basic Examples - GPIO, timers, interrupts
  • Motor Control - FOC, BLDC, sensor interfaces
  • Communication - UART, SPI, I2C, CAN
  • Power Conversion - SMPS, inverters, PFC
How to Access Examples:
  1. In MATLAB, click Home → Open
  2. Navigate to blockset installation folder → examples
  3. Copy example to your working directory before modifying

  3. Add Constant block for duty cycle input

  1. Add ADC block to read analog input
  2. Add UART Tx block for serial output

Troubleshooting

  • Build fails - Check compiler installation and paths
  • Device not found - Verify Device Family Pack (DFP) is installed
  • Code doesn't run - Check oscillator configuration matches hardware
  • Simulation error - Set proper sample times for all blocks
