MCHP_ADC_SAME5x - ADC for SAM E5x/D5x/C2x

ADC_SAMx5_SAMC2x Block Icon

Block Description

The MCHP_ADC_SAME5x block implements the ADC peripheral for Arm Cortex-M4 based SAM E5x, SAM D5x, and SAM C2x families. Features software-programmable sequencing with up to 6 sequences and 6 conversions per sequence, providing flexible multi-channel acquisition.

Supported Devices

  • SAM E5x - High-performance Cortex-M4F (SAME51, SAME53, SAME54)
  • SAM D5x - Cost-optimized Cortex-M4F (SAMD51)
  • SAM C2x - Motor control focused (SAMC20, SAMC21)

Key Features

Feature Description
Resolution 12-bit SAR ADC
Cores Dual ADC (ADC0, ADC1) on SAMC2x, single ADC on SAMD
Sequencing 6 sequences × 6 conversions = 36 programmable slots
Channels Up to 20 single-ended or 10 differential inputs
Speed Up to 1 Msps

Configuration

Sequence-Based Configuration

  • ADC0_seq0 through ADC0_seq5 - Primary ADC sequences
  • ADC1_seq0 through ADC1_seq5 - Secondary ADC sequences (SAMC2x only)
  • Each sequence: Configure up to 6 channel conversions
  • Pin selection: Dropdown shows available ANx pins with port mapping
Note: Pin availability and ADC core count varies by device. The block automatically detects chip capabilities and shows only valid options.

Applications

  • Motor control feedback (position, current, voltage)
  • Multi-channel sensor acquisition
  • Power supply monitoring
  • Industrial automation
