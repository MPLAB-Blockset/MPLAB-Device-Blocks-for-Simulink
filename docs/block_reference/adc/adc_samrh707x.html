MCHP_ADC_SAMRH707x - Radiation-Hardened ADC

ADC_SAMRH707x Block Icon

Block Description

The MCHP_ADC_SAMRH707x block implements the ADC peripheral for the SAM RH707x radiation-hardened Arm Cortex-M7 microcontroller family. Designed for aerospace, satellite, and high-reliability applications requiring fault tolerance and radiation immunity.

Radiation-Hardened: This ADC is designed to operate reliably in high-radiation environments with built-in error detection and correction mechanisms.

Supported Devices

  • SAMRH707 - 300 MHz Arm Cortex-M7 with radiation hardening
  • SAMRH70 - Radiation-tolerant variant

Key Features

Feature Specification
Resolution 12-bit SAR ADC
Speed Up to 1 Msps
Channels Multiple analog inputs (device dependent)
Reliability Radiation hardening, ECC, fault detection
Operating Temperature Extended range for space applications

Configuration

Similar sequence-based configuration to SAM E5x ADC block, with additional reliability features and radiation-aware timing parameters.

Applications

  • Satellite telemetry and control
  • Space instrumentation
  • Nuclear facility monitoring
  • High-altitude aviation systems
  • Military/defense applications
Design Consideration: When using this ADC in radiation environments, follow device-specific guidelines for redundancy, error checking, and periodic recalibration to maintain accuracy.

Related Blocks

  • MCHP_ADC_SAME5x - For commercial SAM E5x devices
  • MCHP_AFEC_SAM7x - For SAM E7x/S7x/V7x families
