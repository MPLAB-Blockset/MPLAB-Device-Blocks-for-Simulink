Communication Blocks

Communication interface blocks for serial, network, and industrial protocols.

UART (Universal Asynchronous Receiver Transmitter)

  • UART Config - Configure UART module (baud rate, pins, implementation mode)
  • UART Tx - Transmit data through UART
  • UART Rx - Receive data from UART
  • UART TxMatlab - Transmit data to MATLAB (picgui interface)
  • UART Break/Autobaud - Break detection and auto-baud features

Other Communication Protocols

  • SPI Master/Slave - Serial Peripheral Interface
  • I2C Master/Slave - Inter-Integrated Circuit
  • CAN Tx/Rx - Controller Area Network
  • CAN-FD - CAN with Flexible Data-rate
  • USB - Universal Serial Bus (PIC32, SAM)
  • Ethernet - 10/100 Ethernet MAC (PIC32MZ, SAME70)
