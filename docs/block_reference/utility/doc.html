MCHP_Doc - Documentation Access Block

Doc Block Icon

Description

The MCHP_Doc block provides quick access to MCHP Blockset documentation, help files, examples, and release notes directly from the Simulink model.

Features

  • One-click access to block help
  • Quick links to user guide
  • Example browser
  • Release notes and changelog
  • Online support resources

No Configuration Required

The MCHP_Doc block has no parameters. Simply double-click to open the documentation menu.
