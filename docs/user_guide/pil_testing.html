Processor-in-the-Loop (PIL) Testing

PIL testing verifies generated code by executing it on actual hardware and comparing results with simulation.

Benefits

  • Verify code generation correctness
  • Measure execution time and performance
  • Detect numerical differences between simulation and hardware
  • Profile code for optimization

PIL Workflow

  1. Create PIL block from subsystem
  2. Configure communication interface
  3. Run simulation with PIL block
  4. Compare results with normal simulation
  5. Analyze performance metrics
